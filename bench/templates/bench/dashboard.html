Novel CI/CD Metrics

Concise definitions used in the evaluation. Longer discussion appears in the dissertation.

  • LCE – Layer Cache Efficiency Share of Docker layers served from cache across runs; approximates how cache-friendly a pipeline is.
  • PRT – Pipeline Recovery Time Time needed to go from a failing pipeline state back to a stable green run.
  • SMO – Secrets Management Overhead Extra latency introduced by fetching, decrypting and injecting secrets into jobs.
  • DEPT – Dynamic Environment Provisioning Time Duration of Terraform / cloud API calls that create or destroy ephemeral environments.
  • CLBC – Cross-Layer Build Consistency Score describing how similar Docker image layers remain across runs and tools (reproducibility).
